# Product Feel

The app should feel like a collaborative studio session, not a database admin panel.

## Core Feeling

- Fast
- Safe
- Musical
- Reversible
- Collaborative
- Calm under conflict

## What The User Should Feel

- "I can try ideas without ruining anything."
- "I can see what my collaborators are doing."
- "I can return to any previous version."
- "The app understands that audio is not text."
- "Conflicts are normal creative forks, not errors."

The DAW should avoid destructive actions. Deletes, trims, fades, merges, pitch shifts, and processing steps should create new versioned states instead of overwriting source material.
