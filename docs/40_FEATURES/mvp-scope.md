# MVP Scope

The MVP should prove that collaborative demo versioning works.

## Included

- Auth
- Groups
- Projects
- Demos
- Track upload
- Recording
- Waveform playback
- Comments
- Annotations
- Version history
- Branching
- Revert
- Basic merge and conflict handling
- Tempo/key detection
- Processing status
- Equipment planning

## Not MVP

- Full MIDI editor
- Full plugin marketplace
- Advanced mastering suite
- Full multitrack mixing console
- Sample library marketplace
- Streaming distribution
- Social network features

This keeps agents from building side quests.
