# Core Metaphor

Git for Music is:

- Google Drive for presence, comments, lightweight metadata, and shared workspace updates
- Git for demo history, branches, reverts, merge logic, and offline sync
- A DAW for timeline editing, tracks, regions, playback, recording, fades, and processing
- A studio assistant for waveform generation, tempo/key detection, vocal detection, transcription, and other background jobs

The app should not become a full Pro Tools clone. The MVP is not about every possible DAW feature. It is about safe collaboration and version history for demos.

This keeps feature creep under control.
